# Eatfit Blog Website

A responsive food blog website created using HTML and CSS.  
This project displays different food items, health blogs, meal plans, and social media links in a modern card layout.

## Features
- Responsive Navbar
- Food Blog Cards
- Attractive UI Design
- Social Media Icons
- Footer Section
- Multiple Food Images and Articles

## Technologies Used
- HTML5
- CSS3
- Font Awesome Icons

## Project Structure
- index.html
- health.css

## Purpose
This project was created for practicing frontend web development and improving UI design skills.
